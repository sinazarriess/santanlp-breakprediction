
Had problems with the installation of the transformers package. Solution: don't install via conda but via pip!

conda install pytorch torchvision torchaudio cudatoolkit=10.2 -c pytorch
pip install transformers

First source bashrc:
> source ~/.bashrc

Use the "bertfinetuning" environment to run this!
> conda activate bertfinetuning
